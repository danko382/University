
\section{Однородные линейные уравнения с постоянными 
    коэффициентами}
    
    Стоит сказать, что Корнев разрешил решать однородные
    линейные уравнения с постоянными коэффициентами
    только методом Эйлера, и никаким больше.

    \subsection{Как выглядит однородное линейное уравнение
    с постоянными коэффициентами?}

        Однородное линейное уравнение с постоянными
        коэффициентами имеет вид:

        \[
            y^{(n)} + a_1 y^{(n - 1)} + \dots +
            a_{n - 1} y^{'} + a_n y = 0,  
        \]
        
        где $a_1, \dots, a_n$~--- заданные числа.

    \subsection{Какие виды однородных линейных уравнений
    с постоянными коэффициентами могут встретиться на
    контрольной работе?}

        Я различаю следующие виды:

        \begin{enumerate}
            \item{Уравнения, имеющие вещественные корни;}
            \item{Уравнения, имеющие комплексныекорни;}
        \end{enumerate}

        Так же стоит заметить, что каждый из приведённых
        выше типов ещё делится на 2 случая:

        \begin{enumerate}
            \item{Случай простых корней;}
            \item{Случай кратных корней.}
        \end{enumerate}

        Дальше мы разберём каждый вид в 2 возможных случаях
        + ещё дополнительно разберём примеры
        из сборника задач, который нам дал Корнев.

    \subsection{Случай вешественных не кратных корней}

        \subsubsection{Теория}
            В данном случае уравнения решаются по следующему
            алгоритму:

            \begin{enumerate}
                \item{Выписываем соответствующее
                характеристическое уравнение. 
                
                Оно будет иметь вид:
                
                \[
                    \lambda^{n} + a_1 \lambda^{n - 1} +
                    \dots + a_{n - 1} \lambda + a_n = 0;
                \]
                }
                \item{Решаем соответствующее характеристическое
                уранение.
                
                После его решения мы получим n-1 простых
                вещественных корней $(\lambda_1, \dots,
                \lambda_{n - 1})$;}
                \item{Выписываем фундаментальную систему решений
                исходного уравнения.
                
                Она будет иметь вид:
                
                \[
                    y_1 = e^{\lambda_1 x}, \dots, y_{n-1} =
                    e^{\lambda_{n - 1} x};    
                \]
                }
                \item{Выписываем общее решение исходного
                уравнения.
                
                Оно будет иметь вид:
                
                \[
                    y = c_1 y_1 + c_2 y_2 + \dots + c_{n - 1}
                    y_{n - 1}. 
                \]
                }
                \item{
                    Записываем ответ.

                    Ответ: $y = \dots$.
                }
            \end{enumerate}
        
        \subsubsection{Практика}

            \textbf{Пример 1:}
            \bigskip

                Дано:

                \[
                    y^{''} + y^{'} - 2 y = 0
                \]

                Решение:
                
                \begin{enumerate}
                    \item{Составляем характеристичное уравнение
                    
                    \[
                        \lambda^2 + \lambda - 2 = 0
                    \]
                    }
                    \item{Решаем характеристическое уравнение

                    \[
                        D = 1 - 4 * (-2) = 3^2
                    \]
                    \[
                        \lambda_1 = \frac{-1 - 3}{2} = -2
                    \]
                    \[
                        \lambda_2 = \frac{-1 + 3}{2} = 1
                    \]
                    }
                    \item{Выписываем ФСР

                    \[
                        y_1(x) = e^{\lambda_1 x} = e^{-2x}
                    \]
                    \[
                        y_2(x) = e^{\lambda_2 x} = e^{x}
                    \]
                    }
                    \item{Выписываем общее решение исходного
                    уравнения

                    \[
                        y = c_1 y_1(x) + c_2 y_2(x) =
                        c_1 e^{-2x} + c_2 e^x   
                    \]
                    }
                    \item{Ответ:
                    $y = c_1 e^{-2x} + c_2 e^x$
                    }
                \end{enumerate}

            \textbf{Пример 2:}
            \bigskip    

                Дано:

                \[
                    y^{(5)} - 10 y^{(3)} + 9 y^{'} = 0
                \]

                Решение:

                \begin{enumerate}
                    \item{Выписываем характеристическое 
                    уравнение

                    \[
                        \lambda^{5} - 10 \lambda^{3} +
                        9 \lambda = 0  
                    \]
                    }
                    \item{Решаем характеристическое уравнение
                    
                    Воспользуемся Схемой Горнера:

                    \begin{tabular}{|c|c|c|c|c|c|c|c|}
                        \hline 
                          & 1 & 0 & -10 & 0 & 9 & 0 & \\ 
                        \hline
                        1 & 1 & 1 & -9 & -9 & 0 & 0 & $\lambda_1 = 1$\\
                        \hline
                        -1 & 1 & 0 & -9 & 0 & 0 &   & $\lambda_2 = -1$\\
                        \hline
                        3 & 1 & 3 & 0 & 0 &   &   & $\lambda_3 = 3$\\
                        \hline
                       \end{tabular}
                    
                    Тогда имеем:

                    \[
                        (\lambda - 1)(\lambda + 1)(\lambda - 3)
                        (\lambda^2 + 3 \lambda) = 0
                    \]

                    Решим $\lambda^2 + 3 \lambda = 0$:
                    \bigskip

                    \[
                       \lambda^2 + 3 \lambda = 0
                    \]
                    \[
                       \lambda (\lambda + 3) = 0
                    \]
                    \[
                       \lambda_4 = 0, \;\;\; \lambda_5 = -3
                    \]

                    Таким образом получим корни:

                    \[
                        \lambda_1 = 1, \;\;\; \lambda_2 = -1,
                        \;\;\; \lambda_3 = 3, \;\;\; \lambda_4 =
                        0, \;\;\; \lambda_5 = -3
                    \]
                    }
                    \item{Выписываем ФСР

                       \[
                       y_1 = e^x, \;\;\; y_2 = e^{-x}, \;\;\;
                       y_3 = e^{3x}, \;\;\; y_4 = e^0 = 1, \;\;\;
                       y_5 = e^{-3x}
                       \]
                    }
                    \item{Выписываем общее решение

                    \[
                       y = c_1 e^x + c_2 e^{-x} + c_3 e^{3x} +
                       c_4 + c_5 e^{-3x}
                    \]
                    }
                    \item{Ответ: $y = c_1 e^x + c_2 e^{-x} +
                    c_3 e^{3x} + c_4 + c_5 e^{-3x}$
                    }
                \end{enumerate}

    \subsection{Случай вещественных кратных корней}  
    
        \subsubsection{Теория}

            В данном случае алгоритм решения будет
            следующий:

            \begin{enumerate}
                \item{Выписываем соответствующее
                характеристическое уравнение. 
                
                Оно будет иметь вид:
                
                \[
                    \lambda^{n} + a_1 \lambda^{n - 1} +
                    \dots + a_{n - 1} \lambda + a_n = 0;
                \]
                }
                \item{Решаем соответствующее характеристическое
                уранение.
                
                После его решения мы получим 1 корень
                $\lambda_1$~--- $n-1$"=кратности;}
                \item{Выписываем фундаментальную систему решений
                исходного уравнения.
                
                Она будет иметь вид:
                
                \[
                    y_1 = e^{\lambda_1 x}, \;\;\;
                    y_2 = x e^{\lambda_1 x}, \;\;\;
                    y_3 = x^2 e^{\lambda_1 x}, \;\;\;
                    \dots,
                    y_{n-1} = x^{n-1} e^{\lambda_{1} x};    
                \]
                }
                \item{Выписываем общее решение исходного
                уравнения.
                
                Оно будет иметь вид:
                
                \[
                    y = c_1 y_1 + c_2 y_2 + \dots + c_{n - 1}
                    y_{n - 1}. 
                \]
                }
                \item{
                    Записываем ответ.

                    Ответ: $y = \dots$.
                }
            \end{enumerate}

            \textbf{У нас в случае кратных корней на практике
            встречались уравнения только с 1 корнем, поэтому,
            скорее всего, других случаев на контрольной 
            работе не встретится.}

        \subsubsection{Практика:}
        
            \textbf{Пример:}
            \bigskip

            Дано:

            \[
                y^{'''} - 3 y^{''} + 3 y^{'} - y = 0  
            \]

            Решение:

            \begin{enumerate}
                \item{Составляем характеристичное уравнение
                
                \[
                    \lambda^3 - 3 \lambda^2 + 3 \lambda - 1 = 0
                \]
                }
                \item{Решаем характеристическое уравнение

                \[
                    \lambda^3 - 3 \lambda^2 + 3 \lambda - 1 = 0
                \]
                \[
                    (\lambda - 1)^3 = 0
                \]
                \[
                    \lambda_1 = 1
                \]
                }
                \item{Выписываем ФСР

                \[
                    y_1(x) = e^{\lambda_1 x} = e^x
                \]
                \[
                    y_2(x) = x e^{\lambda_1 x} = x e^x
                \]
                \[
                    y_3(x) = x^2 e^{\lambda_1 x} = x^2 e^x
                \]
                }
                \item{Выписываем общее решение исходного
                уравнения

                \[
                    y = c_1 y_1(x) + c_2 y_2(x) + c_3 y_3(x) =
                    c_1 e^{x} + c_2 x e^x + c_3 x^2 e^x   
                \]
                }
                \item{Ответ:
                $y = c_1 e^{x} + c_2 x e^x + c_3 x^2 e^x$
                }
            \end{enumerate}

    \subsection{Случай комплексных не кратных корней}
    
        \subsubsection{Теория}

            Алгоритм решения:

            \begin{enumerate}
                \item{Выписываем соответствующее
                характеристическое уравнение. 
                
                Оно будет иметь вид:
                
                \[
                    \lambda^{n} + a_1 \lambda^{n - 1} +
                    \dots + a_{n - 1} \lambda + a_n = 0;
                \]
                }
                \item{Решаем соответствующее характеристическое
                уранение.
                
                После его решения мы получим n-1 простых
                вещественных и комплексных корней
                $(\lambda_1, \dots, \lambda_{n - 1})$;}
                \item{Выписываем фундаментальную систему решений
                исходного уравнения.
                
                Она будет иметь вид:
                
                \[
                    y_1 = e^{\lambda_1 x}, \dots, y_{n-1} =
                    e^{\lambda_{n - 1} x};    
                \]
                }
                \item{Преобразовываем комплексные сопряжённые 
                решения по формуле Эйлера.

                \[
                    e^{\alpha + i \gamma} = e^{\alpha}(\cos{
                    \gamma} + i\sin{\gamma})
                \]

                Для определённости предположим, что
                корни $\lambda_1$ и $\lambda_2$ комплекно
                сопряжённые, тогда из 2, соответствующих им 
                решений, получим 1 решение:

                \[
                    y_1 =  e^{\alpha}(\cos{\gamma} +
                    \sin{\gamma}) = z_1,
                \]

                а решение $y_2$ отбрасываем

                }
                \item{Выписываем общее решение исходного
                уравнения.
                
                Оно будет иметь вид:
                
                \[
                    y = c_1 y_1 + c_2 y_2 + \dots + c_{k}
                    y_{k} + c_{k + 1} z_1 + \dots + c_{k + m} z_m, 
                \]

                где $y_1, \dots, y_k$~--- вещественные корни,
                а $z_1, \dots, z_m$~--- преобразованные
                комплексные корни.
                }
                \item{
                    Записываем ответ.

                    Ответ: $y = \dots$.
                }
            \end{enumerate}

        \subsubsection{Практика:}
        
            \textbf{Пример 1:}
            \bigskip

            Дано:

            \[
                y^{''} + 4 y = 0  
            \]

            Решение:

            \begin{enumerate}
                \item{Составляем характеристичное уравнение
                
                \[
                    \lambda^2 + 4 = 0
                \]
                }
                \item{Решаем характеристическое уравнение

                \[
                    \lambda^2 + 4 = 0
                \]
                \[
                    \lambda = -4
                \]
                \[
                    \lambda_1 = 2 i
                \]
                \[
                    \lambda_2 = - 2 i  
                \]
                }
                \item{Выписываем ФСР

                \[
                    y_1(x) = e^{\lambda_1 x} = e^{2ix}
                \]
                \[
                    y_2(x) = e^{\lambda_2 x} = e^{-2ix}
                \]
                }
                \item{Преобразовываем комплексные решения
                
                Рассматриваем $y_1$, а $y_2$, как комплексно 
                сопряженное $y_1$, отбрасываем:

                \[
                    y_1 = e^{2ix} = e^0 (\cos{2x} + \sin{2x}) =
                    z_1
                \]

                }
                \item{Выписываем общее решение исходного
                уравнения

                \[
                    y = c_1 z_1 = c_1 (\cos{2x} + \sin{2x}) =
                    c_1 \cos{2x} + c_2 \sin{2x}   
                \]
                }
                \item{Ответ:
                $y = c_1 \cos{2x} + c_2 \sin{2x}$
                }
            \end{enumerate}

            \textbf{Пример 2:}
            \bigskip

            Дано:

            \[
                y^{'''} - 8 y = 0  
            \]

            Решение:

            \begin{enumerate}
                \item{Составляем характеристичное уравнение
                
                \[
                    \lambda^3 - 8 = 0
                \]
                }
                \item{Решаем характеристическое уравнение

                Для того, чтобы избавиться от куба
                воспользуемся схемой Горнера:

                \begin{tabular}{|c|c|c|c|c|c|}
                    \hline 
                      & 1 & 0 & 0 & -8 & \\ 
                    \hline
                    2 & 1 & 2 & 4 & 0 & $\lambda_1 = 2$\\
                    \hline
                \end{tabular}

                Таким образом, получим:

                \[
                    (\lambda - 2)(\lambda^2 + 2 \lambda + 4) = 0
                \]

                Решим $\lambda^2 + 2 \lambda + 4 = 0$:
                
                \[
                    D = 4 - 4 * 4 = -12  
                \]
                \[
                    \lambda_2 = \frac{-2 + \sqrt{-12}}{2} =
                    \frac{-2 + 2\sqrt{3}i}{2} = -1 + \sqrt{3}i  
                \]
                \[
                    \lambda_3 = -1 - \sqrt{3}i  
                \]
                }
                \item{Выписываем ФСР

                \[
                    y_1(x) = e^{\lambda_1 x} = e^{2x}
                \]
                \[
                    y_2(x) = e^{\lambda_2 x} = e^{-x +\sqrt{3}ix}
                \]
                \[
                    y_3(x) = e^{\lambda_3 x} = e^{-x -\sqrt{3}ix}
                \]
                }
                \item{Преобразовываем комплексные решения
                
                Рассматриваем $y_2$, а $y_3$, как комплексно 
                сопряженное $y_3$, отбрасываем:

                \[
                    y_2 = e^{-x + \sqrt{3}ix} = e^{-x} (\cos{\sqrt{3}x}
                    + \sin{\sqrt{3}x}) =
                    z_1
                \]

                }
                \item{Выписываем общее решение исходного
                уравнения

                \[
                    y = c_1 y_1 + c_2 z_1 =
                    c_1 e^{2x} + c_2 e^{-x} \cos{\sqrt{3}x} + c_3
                    e^{-x} \sin{\sqrt{3}x}   
                \]
                }
                \item{Ответ:
                $y = c_1 e^{2x} + c_2 e^{-x} \cos{\sqrt{3}x} + c_3
                e^{-x} \sin{\sqrt{3}x}$
                }
            \end{enumerate}

            \textbf{Пример 3:}
            \bigskip

            Дано:

            \[
                y^{(6)} + 64 y = 0
            \]  

            Решение:

            \begin{enumerate}
                \item{Составляем характеристичное уравнение
                
                \[
                    \lambda^6 + 64 = 0
                \]
                }
                \item{Решаем характеристическое уравнение

                \[
                    \lambda^6 + 64 = 0  
                \]
                \[
                    \lambda^6 = -64  
                \]
                \[
                    \lambda = \sqrt[6]{-64}  
                \]
                \[
                    \lambda_1 = \sqrt[6]{64} (\cos{(
                    \frac{\pi}{6})} +
                    i\sin{(\frac{\pi}{6})}) =
                    2 (\frac{\sqrt{3}}{2} + i\frac{1}{2}) = 
                    \sqrt{3} + i
                \]
                \[
                    \lambda_2 = \sqrt[6]{64} (\cos{(
                    \frac{3\pi}{6})} +
                    i\sin{(\frac{3\pi}{6})}) = 2i
                \]
                \[
                    \lambda_3 = \sqrt[6]{64} (\cos{(
                    \frac{5\pi}{6})} +
                    i\sin{(\frac{5\pi}{6})}) = -\sqrt{3} + i
                \]
                \[
                    \lambda_4 = \sqrt[6]{64} (\cos{(
                    \frac{7\pi}{6})} +
                    i\sin{(\frac{7\pi}{6})}) = -\sqrt{3} - i
                \]
                \[
                    \lambda_5 = \sqrt[6]{64} (\cos{(
                    \frac{9\pi}{6})} +
                    i\sin{(\frac{9\pi}{6})}) = -2i
                \]
                \[
                    \lambda_6 = \sqrt[6]{64} (\cos{(
                    \frac{11\pi}{6})} +
                    i\sin{(\frac{11\pi}{6})}) = \sqrt{3} - i
                \]

                Таким образом, имеем 3 комплексно"=сопряжённых
                пары корней:

                \[
                    \lambda_{1, 2} = \pm 2 i
                \]
                \[
                    \lambda_{3, 4} = \sqrt{3} \pm i  
                \]
                \[
                    \lambda{5, 6} = -\sqrt{3} \pm i  
                \]
                }
                \item{Выписываем ФСР
                
                \[
                    y_{1, 2} = e^{\pm 2ix}, \;\;\;
                    y_{3, 4} = e^{\sqrt{3} \pm i}, \;\;\;
                    y_{5, 6} = e^{-\sqrt{3} \pm i}                    
                \]
                }
                \item{Преобразовываем комплексные решения
                
                Рассматриваем $y_1, y_3, y_5$, а $y_2, y_4, y_6$
                отбрасываем.

                \[
                    z_1 = \cos{2x} + \sin{2x}
                \]
                \[
                    z_2 = e^{\sqrt{3}} \cos{x} + e^{\sqrt{3}} \sin{x}  
                \]
                \[
                    z_3 = e^{-\sqrt{3}} \cos{x} + e^{-\sqrt{3}} \sin{x}
                \]
                }
                \item{Выписываем общее решение исходного
                уравнения

                \[
                    y = 
                    c_1 \cos{2x} + c_2 \sin{2x} +   
                    c_3 e^{\sqrt{3}} \cos{x} + c_4 e^{\sqrt{3}} \sin{x} +
                    c_5 e^{-\sqrt{3}} \cos{x} + c_6 e^{-\sqrt{3}} \sin{x}
                \]
                }
                \item{Ответ:
                $y = 
                c_1 \cos{2x} + c_2 \sin{2x} +   
                c_3 e^{\sqrt{3}} \cos{x} + c_4 e^{\sqrt{3}} \sin{x} +
                c_5 e^{-\sqrt{3}} \cos{x} + c_6 e^{-\sqrt{3}} \sin{x}$
                }
            \end{enumerate}

    \subsection{Случай комплексных кратных корней}   
    
        \subsubsection{Теория}

            Алгоритм аналогичен алгоритму решения в 
            случае не кратных комплексных корней. С единственным
            различием в том, что получая кратные корни мы поступаем
            как в алгоритме случае вещественных кратных корней.

        \subsubsection{Практика}
        
            \textbf{Пример:}
            \bigskip

            Дано:

            \[
                y^{(4)} + 2 y^{''} + y = 0  
            \]

            Решение:

            \begin{enumerate}
                \item{Составляем характеристичное уравнение
                
                \[
                    \lambda^4 + 2 \lambda^2 + 1 = 0
                \]
                }
                \item{Решаем характеристическое уравнение

                \[
                    \lambda^4 + 2 \lambda^2 + 1 = 0 
                \]
                \[
                    (\lambda^2 + 1)^2 = 0  
                \]
                \[
                    \lambda = \pm i  
                \]
                \[
                    \lambda_1 = i
                \]
                \[
                    \lambda_2 = -i
                \]
                
                Таким образом, имеем 2 комплексно"=сопряжённых
                корня второй кратности:
                }
                \item{Выписываем ФСР
                
                \[
                    y_{1} = e^{ix}, \;\;\;
                    y_2 = x e^{ix}, \;\;\;
                    y_{3} = e^{-ix}, \;\;\;
                    y_4 = x e^{-ix}, \;\;\;
                \]
                }
                \item{Преобразовываем комплексные решения
                
                Рассматриваем $y_1, y_2$, а $y_3, y_4$
                отбрасываем + не забываем про 
                кратность (на 1 корень по 2 решения).

                \[
                    z_1 = \cos{x} + \sin{x}
                \]
                \[
                    z_2 = x\cos{x} + x\sin{x}
                \]
                }
                \item{Выписываем общее решение исходного
                уравнения

                \[
                    y = 
                    c_1 \cos{x} + c_2 \sin{x} +   
                    c_3 x\cos{x} + c_4 x\sin{x}
                \]
                }
                \item{Ответ:
                $y = 
                c_1 \cos{x} + c_2 \sin{x} +   
                c_3 x\cos{x} + c_4 x\sin{x}$
                }
            \end{enumerate}
